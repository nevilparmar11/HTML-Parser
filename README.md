# HTML-Parser
Parses Egov marks table for internal and external from the page source code.

## How to use ?
- Login into EGOV Account.
- Visit Exam Results page (Internal/External).
- Press ctrl + U. 
- Copy and paste here the source code with proper exam type.
- Press Submit.
